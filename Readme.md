# Weather web app
This weather web application was created to find out the weather conditions in various places in the world. The application shows the weather forecast for every 3 hours of the time for 4 full days and 2 incomplete days. The application shows how many degrees, weather description, wind speed and humidity in a specific period of time, also shows the time of sunrise and sunset for each day. By default, the application shows the weather forecast according to the geolocation of the device on which the application is open, it is also possible to search for the locality where you want to know the weather forecast, for this you should enter the name of the locality in the search menu in the upper right corner. It is also possible to switch the theme by clicking on the theme switcher in the top middle.
